A collection of concrete (or at least partially realized) commands that may be reused by any MSL client application. These commands implement the API of the {@link org.eclipse.gmf.runtime.emf.commands.core.command} package and offer such generic capabilities as "create an EObject" or "reposition an EObject in an EList."
